Projeto 1 - O Caminho do Eco (Escape room acessivel para pessoas com deficiencia visual).

[ Melhor Projeto 1 da amostra tech CESAR 2025.1 ]

Descrição dos artefatos

  • Artefato (Enigma) 1:

    • O artefato emite sons para atrair o jogador ate ele achar um sensor de toque, que ao ser precionado fara com que o artefato pare de emitir sons sinalizando que o jogador deve progredir para a proxima parte.

  • Artefato (Enigma) 2:

    • O artafeto detecta quando o jogodor chega perto, e começa a emitir dois sons diferentes para que o jogador tente descobrir o qual o objetivo, que nesse caso é tentar fazer com que os dois sons tenham a mesmo frquencia por meio de um potenciometro, ao fazer isso o enigma é completo, e mais uma vez os sons param e o jogador vai para o proximo enigma.

  • Artefato (Enigma) 3:

    • O artefato detecta quando o jogador se aproxima e da as estruções para ele poder completar o enigma, onde o jogador tem que apertar butões na sequencia certa, quando isso acontece um compartimento é aberto liberando o acesso a um objeto para que o proximo enigma possa ser resolvido.

  • Artefato (Enigma) 4:

    • O artefato emite sons para atrair o jogador, que ao chegar no totem tem que achar o lugar onde colocar o objeto obtido no enigma anterior, ao fazer isso o jogador é parabenizado.
